ACTIVITIES


Month 1 – Fun Foundations
Goal: Get members comfortable with basic coding and teamwork.

Week 1 – Icebreaker & Code Challenge
Introduce members, share coding interests.
Mini activity: Create a “Hello, World!” in different languages.


Week 2 – HTML & CSS Workshop
Build a simple personal webpage.
Add fun styling (colors, fonts, backgrounds).


Week 3 – Bug Hunt
Give members broken code snippets.
Teams compete to fix the most bugs.


Week 4 – Creative Coding: ASCII Art
Make pictures using only text characters.
Vote on funniest or most creative.
